## 🚀 Installing Portfolio 🚀
To use, follow the steps:

#### Clone repository
```bash
git clone https://github.com/heyliceeee/portfolio.git
```

#### Install dependencies
```bash
npm i
```

#### Run app
```bash
npm start
```
